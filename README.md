# Data-Driven Surface

## Project Overview
This project explores the transformation of literary spatial imagination into architectural form using a data-driven workflow.

## Data Sources
- Invisible Cities (text)
- Flickr API (image dataset)
- Derived captions / embeddings

## Workflow
1. Text processing
2. Image dataset construction
3. Cross-domain embedding & retrieval
4. Clustering & mapping
5. Spatial generation
